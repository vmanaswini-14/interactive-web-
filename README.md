🌍 InAmigos Foundation – NGO Website

A responsive and modern NGO website built using HTML and CSS, designed to showcase the mission, projects, impact, and contact details of the InAmigos Foundation.

📌 Project Overview

This project is a static website developed to represent a non-profit organization. It includes multiple sections that highlight the foundation’s vision, ongoing projects, achievements, and ways for users to connect.

The goal of this project is to demonstrate frontend development skills, layout structuring, and responsive design using pure HTML and CSS.

🚀 Features
🏠 Modern landing (Hero) section
📖 About Us (Vision, Mission, Values)
📂 Projects showcase section
📊 Impact statistics section
🖼️ Interactive gallery with popup view
📞 Contact form UI
🔗 Social media links
📱 Fully responsive design (mobile-friendly)
🎨 Smooth hover effects and modern UI styling
🛠️ Tech Stack
HTML5
CSS3 (Flexbox, Media Queries, Animations)
Font Awesome Icons
